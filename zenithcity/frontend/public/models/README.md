# 3D Building Models

Place your Kenney.nl City Kit GLB files here:

| Filename       | Source pack              | URL                                        |
|----------------|--------------------------|--------------------------------------------|
| house.glb      | City Kit (Suburban)      | https://kenney.nl/assets/city-kit-suburban   |
| apartment.glb  | City Kit (Industrial)    | https://kenney.nl/assets/city-kit-industrial |
| office.glb     | City Kit (Commercial)    | https://kenney.nl/assets/city-kit-commercial |
| park.glb       | City Kit (Suburban)      | https://kenney.nl/assets/city-kit-suburban   |
| stadium.glb    | City Kit (Commercial)    | https://kenney.nl/assets/city-kit-commercial |

All packs are **CC0** (public domain) — free for commercial use.

Or run the auto-downloader:
```
cd frontend && bash scripts/download-models.sh
```

The app works without these files — a stylised fallback renders for any missing model.
